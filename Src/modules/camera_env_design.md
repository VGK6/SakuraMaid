二、CameraEnv 模块设计
━━━━━━━━━━━━━━━━━━━━

┌──────────────────────────────────────┐
│  CameraEnv                            │
│                                       │
│  __init__()                           │
│  ├─ self.cap = cv2.VideoCapture(0)    │
│  ├─ self.face_cascade = HaarCascade   │
│  └─ self.last_frame = None            │
│                                       │
│  capture() → np.ndarray               │
│  └─ 读取摄像头一帧                     │
│                                       │
│  detect_faces(frame) → int            │
│  └─ 返回检测到的人脸数量               │
│                                       │
│  detect_motion(frame) → bool          │
│  └─ 帧差法检测是否有运动               │
│                                       │
│  get_scene() → dict                    │
│  ├─ face_count: 人脸数                │
│  ├─ has_motion: 是否有运动             │
│  ├─ is_dark: 是否光线暗(深夜)         │
│  └─ brightness: 画面亮度值            │
│                                       │
│  release()                            │
│  └─ 释放摄像头资源                     │
└──────────────────────────────────────┘


三、数据流
━━━━━━━━━━━━━━━━━━━━

ProactiveEngine._loop (每30秒)
       │
       ▼
CameraEnv.get_scene()
       │
       ├─ face_count = 0, is_dark = True
       │    → 深夜无人 → 跳过
       │
       ├─ face_count = 0, is_dark = False
       │    → 白天无人 → 无操作计时++
       │    → 超过2小时 → "还在吗？"
       │
       ├─ face_count = 1, h >= 0 and h <= 5
       │    → 深夜有人 → "该休息了！"
       │
       ├─ face_count >= 2
       │    → 多人在场 → 切换安静模式
       │    → 气泡提示改为静默
       │
       └─ face_count >= 1, has_motion = True
            → 正常使用 → 重置无操作计时

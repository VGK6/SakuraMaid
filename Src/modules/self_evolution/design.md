┌─────────────┐     ┌─────────────┐     ┌─────────────┐

│ (内置/快速) │     │  + 沙盒测试  │     │  + 经验库   │
└──────┬──────┘     └──────┬──────┘     └──────┬──────┘

                          ▼

                          ▼

                  └───────────────┘

二、数据库表
━━━━━━━━━━━━━━━━━━━━

【skills】技能表
  id, name, trigger_keywords, code, version, priority,
  enabled, health_score, env_fingerprint, created_at, updated_at

【skill_aliases】技能别名表
  skill_id FK, alias

【skill_logs】执行日志表
  id, skill_id FK, success, duration_ms, error_hash, env_fingerprint, timestamp

【fix_experience】修复经验表
  id, error_hash, error_template, solution_code, fix_count, created_at

三、核心流程
━━━━━━━━━━━━━━━━━━━━
  匹配 → 生成 → 执行 → 诊疗 → 修复

四、安全沙盒
━━━━━━━━━━━━━━━━━━━━
  静态扫描 + 路径白名单 + 二次确认 + 执行限制

五、原子技能库
━━━━━━━━━━━━━━━━━━━━
  内置基础技能，不占用LLM token
